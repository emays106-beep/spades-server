# Spades Multiplayer Server

Real-time multiplayer Spades game server built with Node.js and WebSockets.

Supports:
- Public matchmaking
- Private rooms (invite code)
- Real-time gameplay
- Scoring engine
- Reconnect support

Built for a Flutter mobile client (iOS & Android).

## Tech Stack
- Node.js
- TypeScript
- WebSocket (ws)
- Real-time game engine

## Run locally

```bash
npm install
npx ts-node src/index.ts
ws://localhost:8080
